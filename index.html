About me

Aditya is motivated by the belief that data can be used to make better decisions. He wants to be able to make it easier for people to understand code and use data efficiently. Aditya believes in open data and open source software. Aditya primarily uses Python, but also has experience with R and Scala.

Aditya has worked at leading cyber security companies like Palo Alto Networks, Cloud Raxak and McAfee on network security, endpoint security and configuration and compliance. Aditya graduated from the School of Information at UC Berkeley where he explored various facets of data science, data visualization and user research.
